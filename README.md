## Installing Go 
To install Go programming langauge see [Install Golang](https://golang.org/doc/install).

## Running a Go program
```
cd arrays/
go run main.go 
```

## Building a Go program
Build these following Go programs before running 
- arguments
- flags

For Example:
```
cd arguments/
go build main.go
./main a b c   
```
